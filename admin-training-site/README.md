# Black Russia - Система подготовки администрации

## 📋 Описание
Полнофункциональный сайт для тестирования и обучения администраторов проекта Black Russia.

## 🎯 Функции

### Категории вопросов:
- ✅ Правила сервера (20 вопросов)
- ✅ Правила администрации (20 вопросов)
- ✅ РП Термины (10 вопросов)
- ✅ Все вопросы вместе (50 вопросов)

### Режимы тестирования:
1. **Выбор ответов** - Выберите правильный ответ из предложенных вариантов
2. **Текстовый ответ** - Напишите ответ своими словами
3. **Наказание** - Выберите тип наказания и уточняющие данные

## 🎨 Интерфейс
- Стилизация в цветах BLACK RUSSIA (черный, красный, неоновый)
- Адаптивный дизайн для всех устройств
- Анимированный интерфейс
- Модальные окна с подробной информацией

## 📊 Статистика
- Отслеживание количества вопросов
- Счётчик правильных ответов
- Процент правильных ответов
- Визуальный прогресс-бар

## 🔧 Возможности
- 💡 Подсказки к каждому вопросу
- ℹ️ Полная информация о правилах
- 🔄 Сброс прогресса
- 📈 Показ результатов
- 🎯 Навигация по вопросам

## 🚀 Как использовать
1. Откройте index.html в браузере
2. Выберите категорию вопросов
3. Выберите режим ответов
4. Начните тестирование
5. Используйте подсказки при необходимости
6. Просмотрите результаты
